# 🧠 AI Resume Builder

AI Resume Builder is a full-stack web application that helps users generate, edit, and export professional resumes using AI-powered suggestions.  
It is designed as a **SaaS-style project for portfolio and internship applications**, focusing on real-world product experience.

---

## 📌 Overview

This project allows users to:
- Create resumes step-by-step
- Generate AI-powered bullet points for experience/projects
- Customize resume theme colors
- Live preview resume in real time
- Export resumes as PDF

Built with a focus on **clean UI, scalable architecture, and production-level practices**.
